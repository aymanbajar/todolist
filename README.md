# 📝 ToDo List App

A simple and efficient ToDo List application built with **React + Vite**, utilizing modern React Hooks (`useState`, `useEffect`, `useContext`, `useReducer`, `useMemo`) for state and logic management.

## 🚀 Features

- ✅ Add new tasks  
- 🧾 View all tasks (completed & not completed)  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as completed  
- ♻️ State management using React Hooks  

## 🛠️ Technologies Used

- **React** (with Vite)
- **React Hooks**: `useState`, `useEffect`, `useContext`, `useReducer`, `useMemo`
- **HTML5**, **CSS3**, **JavaScript (ES6+)**
